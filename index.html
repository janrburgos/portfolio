Deal or No Deal

A JavaScript game based on the tv show Deal or No Deal. The player first chooses his/her briefcase and then choose 6 other briefcases to open on the first round. After opening 6 briefcases, the banker gets the average of values of unopened briefcases and offers it to the player. If the player presses "Deal", he/she gets the offered value, if he/she presses "No Deal", he/she proceeds to the next round but this time only 5 briefcases should be opened before the banker gives an offer again. The number of briefcases to be opened reduces every round.

To Do App

A to do app created using create-react-app and Redux. The task name written on the input gets added to the pending tasks when the add task button is pressed. Also, there will be an error message shown if the task name you are trying to add is already existing in either the pending or done tasks. If you hover on each item on the pending tasks, buttons will show which will either delete the task or move the task to the done task group. You can also delete the task item in the done tasks.
